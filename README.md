# 🎨 피규어 주문 관리 앱

## 실행 방법

### 1단계 — Node.js 설치 확인
터미널(cmd)을 열고 아래 명령어 입력:
```
node -v
```
버전 숫자가 나오면 OK. 안 나오면 https://nodejs.org 에서 설치 (LTS 버전)

### 2단계 — 이 폴더로 이동
```
cd figure-manager
```

### 3단계 — 패키지 설치 (최초 1회만)
```
npm install
```
(2~3분 걸릴 수 있어요)

### 4단계 — 실행
```
npm start
```
브라우저에서 http://localhost:3000 이 자동으로 열립니다!

---

## 데이터 저장 위치
브라우저 로컬스토리지에 자동 저장되므로 껐다 켜도 데이터가 유지됩니다.
